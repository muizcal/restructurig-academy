# Restructuring Academy Website

## Setup Instructions

1. **Extract the zip file** into a folder on your computer.

2. **Add Your Media Files** — Place the following files inside the `assets/` folder:
   - `founder.jpg` — Photo of Princess Amarachi (founder)
   - `sandra.jpg` — Photo of Sandra Nwaekpe
   - `sandra-video.mp4` — Sandra's testimonial video
   - `logo.jpg` — Restructuring Academy logo

3. **Open the website** — Double-click `index.html` in your browser to preview locally.

4. **To go live**, upload all files to your web hosting provider (keep the folder structure intact):
   ```
   restructuring-academy/
   ├── index.html
   ├── assets/
   │   ├── founder.jpg
   │   ├── sandra.jpg
   │   ├── sandra-video.mp4
   │   └── logo.jpg
   ├── css/
   │   └── style.css
   └── js/
       └── main.js
   ```
